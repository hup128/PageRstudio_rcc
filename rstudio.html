Overview

rstudio is a powerful widely used Integrated Developement Environment (IDE) for R. rstudio, being an IDE for R therefore provides a tailored editor for R, a Multiple Document Interface (MDI) where there may be several scripts opened in different tabs, a window that interacts with the compiler, a debugger, file browser and more. It provides better control over various aspects of R, including plotting, file management, command and data management.

Setup

Usage in Linux

Load the module before you start working with rstudio.

module load rstudio

The latest most recent version of rstudio is set as the default. However, there might be other available versions as well, which can be checked by the following command:

module avail rstudio

Usage

Interactive usage: To start rstudio, type:

rstudio

rstudio is a an interactive developement environment for R. The scripted could be later saved as an R file and be excuted in batch. See a sample R PBS script.

For more information on submitting to the queue, check the appropriate system User Guide.

Usage in Linux

Documentation

Documentation of rstudio can be found in the installation directory. Use an editor to read them.

vim /usr/global/rstudio/0.96.331/README.md
